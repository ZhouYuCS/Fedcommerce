# Project README

## Running the Code

To run the code, please first download the Cifar100 datasetï¼Œ or a down-sampled version of the ImageNet dataset from [Image-Net](https://image-net.org/download-images), and specify the data location in `config.py`.

You can obtain the backbone model by executing

`
python base_train.py
`

Then, to start Fedcommerce, simply run

`
python main.py
`

Please ensure that the data path is correctly set  before proceeding.
